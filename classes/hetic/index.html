Installation de Ruby

Afin de pouvoir commencer les TPs rapidement et ainsi profiter au mieux du temps de cours, je vous encourage très vivement à installer Ruby sur vos machines.

MacOS

Il est référable d'installer XCode et les command line tools pour être sûr d'avoir accès à toutes les fonctionnalités de MacOS. Attention, l'installation d'XCode peut être longue.

Pour ce qui est de Ruby, sur Mac vous devriez avoir une version du langage déjà d'installée. Pour vérifier, lancez le Terminal et tapez ruby -v pour obtenir la version exacte. Idéalement vous voulez la version 2.2.x (2.2.0, 2.2.1 etc.), mais toute version supérieure à 2 est acceptable.

Si vous n'avez pas une version valide (1.9.7 par exemple), rendez vous ici: Installez vos outils

Linux

Sur Linux il est possible que Ruby soit déjà installé. Pour vérifier, lancez votre console et exécutez ruby -v pour obtenir la version. Idéalement vous voulez la version 2.2.x (2.2.0, 2.2.1 etc.), mais toute version supérieure à 2 est acceptable.

Si vous n'avez pas une version valide (1.9.7 par exemple), suivez la procédure adaptée à votre distro .

Windows

Il est tout à fait possible de développer en Ruby sous Windows, même si l'expérience reste meilleure sur un système UNIX.

Pour installer simplement, utilisez RubyInstaller et choisissez la version de Ruby 2.1.x (c'est à dire 2.1.0 ou 2.1.1 etc.). N'oubliez pas de cocher "Add Ruby executables to your PATH".

Pour plus d'infos vous pouvez vous rendre sur ce tutoriel .

Ce n'est pas requis, mais il peut être intéressant d'installer un remplacement à la console par défaut de Windows. Plusieures options sont détaillées dans ce post StackOverflow: Is there a better Windows Console Window?
